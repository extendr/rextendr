# Generate LICENSE.note file.

LICENSE.note generated by this function contains information about all
recursive dependencies in Rust crate.

## Usage

``` r
write_license_note(path = ".", quiet = FALSE, force = TRUE)
```

## Arguments

- path:

  character scalar, the R package directory

- quiet:

  logical scalar, whether to signal successful writing of LICENSE.note
  (default is `FALSE`)

- force:

  logical scalar, whether to regenerate LICENSE.note if LICENSE.note
  already exists (default is `TRUE`)

## Value

text printed to LICENSE.note (invisibly).

## Examples

``` r
if (FALSE) { # \dontrun{
write_license_note()
} # }
```
