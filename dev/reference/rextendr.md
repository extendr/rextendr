# Call Rust code from R using the 'extendr' Crate

The rextendr package implements functions to interface with Rust code
from R. See
[`rust_source()`](https://extendr.github.io/rextendr/dev/reference/rust_source.md)
for details.

## See also

Useful links:

- <https://extendr.rs/rextendr/>

- <https://github.com/extendr/rextendr>

- Report bugs at <https://github.com/extendr/rextendr/issues>
